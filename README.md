# Secure PKI Chat Tool

A secure, end-to-end encrypted chat application leveraging Public Key Infrastructure (PKI) for authentication, data integrity, and confidentiality.

## 🚀 Features
- **Certificate Authority (CA):** Centralized trust and certificate issuance.
- **Hybrid Encryption:** AES-256-GCM for message confidentiality, RSA/ECC for key exchange.
- **Digital Signatures:** Ensures message integrity and non-repudiation.
- **Virtual HSM:** Secure, password-protected private key storage (PKCS#12).
- **Attack Mitigation:** Defends against MITM, Replay attacks, and ensures Forward Secrecy.

## 🛠️ Setup & Installation
1. Clone the repository: `git clone <repo-url>`
2. Create a virtual environment: `python3 -m venv venv && source venv/bin/activate`
3. Install dependencies: `pip install -r requirements.txt`
